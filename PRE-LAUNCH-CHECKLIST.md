## üÜò COMMON ISSUES & FIXES

### Issue: Images not showing
**Fix:** Verify image filenames match exactly (case-sensitive) and are in `images/` folder

### Issue: Mobile menu not working
**Fix:** Ensure script.js is uploaded and linked correctly

### Issue: Pages look wrong
**Fix:** Clear browser cache (Ctrl + Shift + R) and try again

### Issue: Forms not submitting
**Fix:** Set up form backend (Formspree, Netlify Forms, or custom)

### Issue: Site not updating
**Fix:** Clear CDN cache if using one, or wait 5-10 minutes
