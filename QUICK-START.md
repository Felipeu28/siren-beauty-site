## Common Issues & Fixes

### Images don't show
- Make sure image files are in `images/` folder
- Check filenames match exactly (case-sensitive)
- Verify images were uploaded with HTML files

### Mobile menu doesn't work
- Ensure `script.js` uploaded correctly
- Clear browser cache
- Check JavaScript isn't blocked

### Pages are blank
- Check all files uploaded to root directory
- Verify file extensions (.html, .css, .js)
- Look for errors in browser console (F12)
